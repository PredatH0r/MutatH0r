InstaBounce

Stingray bounces players with either primary or secondary fire and tags them to be killed with the other fire mode.
After 2 seconds or when a player touches the ground he becomes untagged again, so they can only be killed mid-air.
Players get a green tag when they are hit with the beam. They can then be killed with a plasma ball.
Players get a yellow tag then when they are hit with 2 balls. They can then be killed with a beam.
In SquadAssault you can untag your team mates with the shot that would normally kill them (green: ball, yellow: beam).

Roq3t

Modifed rocket damage, knockback and fire rate.

Config

KnockbackFactorHoriz, KnockbackFactorVert: Modifies the knockback from a rocket hit

MinKnockbackVert, MaxKnockbackVert: Min/Max vertical knockback

DamageFactorDirect: direct hit damage (100) is multiplied with this factor (e.g. 1.2 => 120 dmg)

DamageFactorSplash: splash damage is multiplied with this factor (e.g. 0.5)

DamageFactorSelf: self damage is multiplied with this factor (e.g. 0.0)

FireInterval: time it takes to reload in seconds (e.g. 0.85)

Regen/Decay

Allows to increase/decrease health and armor over time.

Typical use cases are health/armor decay (if they are above 100/50) or automatic health regeneration.

Config

TickInterval: Interval in seconds between automatic updates to health and armor

HealthRegenAmount, HealthRegenLimit: if a player's health is below the limit, he gains the specified amount, up to that limit.

HealthDecayUpperAmount, HealthDecayUpperLimit: if a player's health is above the limit, he loses the specified amount, down to that limit.

HealthDecayLowerAmount, HealthDecayLowerLimit: if a player's health is above the limit, he loses the specified amount, down to that limit.

ArmorRegenAmount, ArmorRegenLimit: if a player's armor is below the limit, he gains the specified amount, up to that limit.

ArmorDecayAmount, ArmorDecayLimit: if a player's armor is above the limit, he loses the specified amount, down to that limit.

 

Note: An Amount value of 0 disables a particular regen/decay setting.

Stim Head

Extra health and armor makes you an easier target for head shots.

Scales a player's head size by (health + armor)/100.

Pulse Head

Pulses a player's head size over time so you can get easier head shots, if you can time it right.

Config

PulseTimerInterval: Interval (in sec) how often the size is updated. Default is 0.01

PulsePeriod: Number of pulse timer ticks for a full inflate/deflate rotation. Default is 300

PulseFactor: Base for the size change in the range between 1-4. 2 means that the head can be between 1/2x and 2x the normal size.
